# Cambio visual: HUD compacto

Se redujo y movió el panel técnico de InkaRise EE3 para que no tape la zona de juego.

Cambios realizados:

- Panel principal más pequeño y transparente.
- Nueva ubicación: parte superior izquierda, pero sin ocupar tanta pantalla.
- Información resumida en menos líneas.
- Se agregó tecla **I** para ocultar/mostrar los datos técnicos durante la partida.
- Se mantienen visibles los datos importantes: vidas, puntaje, nivel, objetivo, habilidades y métricas inferiores.

Uso:

- **I**: ocultar/mostrar panel técnico.
- **P** o botón de pausa: pausar/continuar.
- **A**: rebote.
- **F**: rayo.
- **S**: guía.
