# Guion para video demo EE3 - 60 a 90 segundos

## 0-10 segundos: inicio
Mostrar el juego abierto y el HUD. Decir que es InkaRise EE3, un prototipo trigonométrico con combate avanzado.

## 10-25 segundos: disparo base y variables
Mover el mouse para mostrar el ángulo θ en el HUD. Hacer un disparo simple y un spread para recordar la base de EE2.

## 25-45 segundos: ricochet
Apuntar a un muro dorado y presionar `A`. Mostrar que el proyectil rebota. Repetir hacia otro muro para evidenciar distintos ángulos. Mencionar que se usa la normal de colisión.

## 45-60 segundos: RayCast2D/equivalente
Apuntar a un objetivo y presionar `F`. Mostrar el rayo y el resultado del HUD. Explicar que detecta si un objetivo está dentro del alcance.

## 60-75 segundos: homing/seek
Presionar `S` para lanzar el proyectil perseguidor. Mostrar cómo corrige su trayectoria hacia el objetivo usando `turn_speed`.

## 75-90 segundos: cierre
Mostrar el HUD con métricas. Decir que el proyecto incluye registro de pruebas, clasificación de variables y limpieza de datos.
