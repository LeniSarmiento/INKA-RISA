# Guion para video demo de 45 a 60 segundos

1. Mostrar pantalla del juego con el personaje inca y el escenario 2D.
2. Decir: "Este es InkaRise v3, un prototipo 2D con disparo trigonométrico y niveles progresivos".
3. Mover el mouse y mostrar cómo cambia el ángulo θ en el HUD.
4. Hacer varios disparos simples con click izquierdo en diferentes direcciones.
5. Usar click derecho para mostrar el patrón angular spread controlado por Δθ.
6. Presionar Q/E para cambiar Δθ y mostrar cómo el abanico se abre o se cierra.
7. Presionar N para enseñar que existen niveles del 1 al 10.
8. Mostrar que en niveles altos los objetivos son más rápidos, pequeños y resistentes.
9. Enfocar el HUD donde aparecen intentos, impactos, fallos y precisión.
10. Cerrar diciendo: "El proyecto aplica seno, coseno y variación angular para crear una mecánica jugable en Godot".
